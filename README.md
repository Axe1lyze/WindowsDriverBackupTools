# WindowsDriverBackupTools

# How to Use Driver Management Batch Files

Please run each batch file with **administrator privileges**.

## 1. export_drivers.bat
Navigates to the directory where the batch file is located and exports all installed drivers to the `exportet_drivers` folder.

## 2. import_drivers.bat
Navigates to the directory where the batch file is located and installs all drivers from the `exportet_drivers` folder within that directory.



# ドライバ管理用バッチファイルの使用方法

各バッチファイルは、**管理者権限**で実行してください。

## 1. export_drivers.bat
バッチファイルが配置されているディレクトリへ移動し、インストール済みのドライバを一括で `exportet_drivers` フォルダへエクスポートします。

## 2. import_drivers.bat
バッチファイルが配置されているディレクトリへ移動し、同ディレクトリ内の `exportet_drivers` フォルダに格納されているドライバを一括でインストールします。
